## Given 2 numbers, A and B, Output TRUE if one of them is 20 or if their sum is 20.

```py

def Makes20(A,B):
    if A ==20 or B == 20 or A+B ==20:
        result = "TRUE"
    else:
        result = "FALSE"
    return result
output= Makes20(19,1)
print(output)

```
